# reputation-pool-cloud

공개 OSS [reputation-pool](https://github.com/PreAgile/reputation-pool)을 엔진으로 쓰는
**호스티드 SaaS의 비공개 상용부**.

- 무엇을/왜/언제는 전부 [PRODUCT.md](PRODUCT.md)에.
- 이 레포는 한글로 운영한다 (문서·이슈·커밋). 코드 식별자는 영어.
- 핵심 규칙: 공개 레포를 fork하지 않고 Maven Central 아티팩트를 의존한다.
  엔진 개선은 공개 레포로, 판매 장치는 여기로.

현재 상태: **Phase 0** — 문서만 존재. 코드는 공개 레포의 core 0.2.0 릴리스 후 시작.
